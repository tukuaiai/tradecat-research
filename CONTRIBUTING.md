# 贡献指南

感谢你为 TradeCat Research 贡献资料。这个仓库的核心目标是让量化交易知识可追溯、可验证、可复用。

## 贡献类型

- 修正文档错误、失效链接或术语不一致。
- 新增数据源说明、研究笔记、策略卡片、实验记录。
- 补充论文、书籍、工具和公开资料索引。
- 改进文档结构、模板、发布流程或质量门禁。

## 提交要求

1. 一个 Pull Request 只处理一个主题。
2. 新增研究内容时优先使用 `docs/contribution-templates/` 下的模板。
3. 引用外部资料时必须给出来源链接、访问日期或版本信息。
4. 涉及策略收益、回测结果或实盘经验时必须说明样本期、费用假设、滑点假设和失效风险。
5. 不提交专有数据、付费数据、未授权转载内容、账户信息或任何敏感凭据。

## 本地预览

```bash
python3 -m venv .venv
. .venv/bin/activate
pip install -r requirements.txt
mkdocs serve
```

## 文档审阅清单

- 标题是否清楚表达主题。
- 结论是否有来源、数据或实验记录支撑。
- 风险边界是否写清楚。
- 链接是否可访问。
- 文档是否已经加入 `mkdocs.yml` 导航。
