# 知识地图

量化研究可以拆成七个互相约束的层次：

1. 市场基础：理解交易机制、订单类型、撮合、流动性和交易成本。
2. 数据基础：确认数据来源、字段语义、清洗规则、复权和幸存者偏差。
3. 研究方法：提出假设、设计实验、控制样本外验证和多重检验风险。
4. 策略逻辑：明确 alpha 来源、持仓规则、调仓规则和失效条件。
5. 风险控制：管理仓位、杠杆、回撤、集中度、相关性和极端行情。
6. 执行落地：处理滑点、冲击成本、订单拆分、限价和成交概率。
7. 工程复现：用版本、日志、实验记录和自动化门禁保持研究可复现。

## 资料进入仓库前的最低要求

- 有清楚主题。
- 有来源或实验记录。
- 有适用边界。
- 有风险提醒。
