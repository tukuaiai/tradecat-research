# 学习框架索引

本页把入门层的学习框架、知识地图和实践路线集中到一起，方便按不同阅读目的进入。

## 学习主线

| 入口 | 适合场景 | 下一步 |
|---|---|---|
| [量化交易学习集合](quant-trading-learning-collection.md) | 想知道量化交易该学什么、先学什么、哪些暂时不学。 | 从领域总览读到 30 天最小可见成果。 |
| [量化交易知识点线面体](quant-trading-knowledge-system.md) | 想把概念、关系、模块和系统串成一个完整认知框架。 | 用“知识点 -> 知识线 -> 知识面 -> 知识体”复盘一个策略想法。 |
| [知识地图](knowledge-map.md) | 想快速看到知识库的整体分层。 | 按市场、数据、研究、策略、风险、执行、工程复现逐层阅读。 |

## 最小行动路线

1. 先读 [量化交易学习集合](quant-trading-learning-collection.md)，确定学习范围。
2. 再读 [量化交易知识点线面体](quant-trading-knowledge-system.md)，建立结构化框架。
3. 进入 [研究流程](../research/research-process.md) 和 [回测框架](../research/backtesting.md)，把学习转成可验证研究。
4. 使用 [研究笔记模板](../contribution-templates/research-note.md) 或 [策略卡片模板](../contribution-templates/strategy-card.md) 输出第一份可审阅作品。
