# 数据字典

## 字段模板

| 字段 | 类型 | 单位 | 含义 | 来源 | 质量约束 |
|---|---|---|---|---|---|
| `symbol` | string | 无 | 标的代码 | 待补充 | 不允许为空 |
| `timestamp` | datetime | UTC 或指定时区 | 数据时间 | 待补充 | 必须单调可排序 |
| `open` | float | 报价货币 | 开盘价 | 待补充 | 必须大于 0 |
| `high` | float | 报价货币 | 最高价 | 待补充 | 必须大于等于 `low` |
| `low` | float | 报价货币 | 最低价 | 待补充 | 必须小于等于 `high` |
| `close` | float | 报价货币 | 收盘价 | 待补充 | 必须大于 0 |
| `volume` | float | 标的单位 | 成交量 | 待补充 | 不应为负 |
