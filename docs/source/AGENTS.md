# AGENTS.md

`docs/source/` 是外部素材的原文拆分存档区。这里的文件用于保留来源材料的完整切片，不承担原创知识正文职责。

## 目录结构

```text
source/
├── AGENTS.md
└── quant-popular-science/
    ├── index.md
    ├── 01-AI量化编程该怎么学-？.md
    ├── 02-量化策略英雄池.md
    ├── 03-量化私募的两种模式.md
    ├── 04-交易的部署模式.md
    ├── 05-如何用期权合成期货.md
    ├── 06-通俗理解期权平价公式.md
    └── 07-算法交易：TWAP、VWAP-与-POV.md
```

## 职责边界

- `quant-popular-science/` 保存 `新建 文本文档.txt` 按 `##` 主题拆出的完整切片。
- `index.md` 只作为拆分目录，不改写素材内容。
- 各切片文件保留原文主题结构，内部 `###`、`####` 不再继续拆分。

## 维护规则

- 该目录只做机械拆分、来源保留、文件归档和导航挂载。
- 不在该目录内改写、润色、压缩或重组原文。
- 如果要把素材转成原创知识正文，应复制到 `docs/foundations/`、`docs/strategies/`、`docs/engineering/` 等正式目录后另行改写。
