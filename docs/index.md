# TradeCat Research

这是一个量化交易研究知识库，用来组织市场基础、数据工程、策略研究、回测验证、风险控制和研究工程化资料。

!!! warning "重要声明"
    本知识库仅用于研究、教育和工程交流，不构成投资建议或交易建议。历史表现不代表未来结果，回测结果不代表实盘收益。

## 阅读路径

1. 从 [知识地图](getting-started/knowledge-map.md) 理解全局结构。
2. 在 [基础](foundations/index.md) 补齐市场、统计和时间序列概念。
3. 进入 [数据](data/index.md) 和 [研究](research/index.md) 建立可复现研究流程。
4. 使用 [策略模板](contribution-templates/strategy-card.md) 记录策略假设、实现方式和风险。
5. 用 [风险](risk/index.md) 与 [合规](compliance/index.md) 检查研究边界。

## 写作原则

- 事实和观点分开。
- 结论和证据绑定。
- 策略和风险同页出现。
- 研究和执行保持边界清晰。
